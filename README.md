# Power BI Sales Dashboard

This project is an interactive Sales Dashboard built using Power BI.

## Key Insights
- Total Sales: 195.98K
- Total Profit: 20.44K
- Profit Percentage: 12%

## Dashboard Features
- Sales trend by month
- Sales by product
- Sales by category
- Sales by payment mode
- Daily sales analysis
- Interactive filters for year and months

## Tools Used
- Power BI
- Excel

## Project Objective
The objective of this dashboard is to analyze sales performance, identify trends, and provide insights for better business decisions.
